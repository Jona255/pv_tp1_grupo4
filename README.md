# Las paginas html fueron diseñadas para organizar y mostrar proyectos académicos universitarios.
#
# Dichos html son:
#
# Inicio: Un panel que muestra estadísticas básicas (proyectos, tareas) y un muro de novedades recientes de los usuarios.
#
# Proyectos: Un listado de proyectos donde se busca trabajos mediante filtros (categoría, año, estado) en un aside y muestra tarjetas con un resumen de cada proyecto.
#
# Detalle: Aqui se puede ver con detalle los objetivos, enlaces a recursos externos (Drive, GitHub) y los integrantes del equipo.
#
# Perfil: Pagina personal del usuario, que incluye datos académicos de la carrera, habilidades técnicas, historial de participación en proyectos y un formulario para actualizar su información.
