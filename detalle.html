Descripción del proyecto

El proyecto tiene como objetivo modernizar y agilizar el proceso de control de asistencia en instituciones educativas. Mediante el uso de códigos QR para los estudiantes, se busca reducir errores humanos, optimizar el tiempo de registro y garantizar una mayor precisión.

El sistema pretende fomentar la transparencia y la seguridad en el manejo de datos estudiantiles. Al integrar la lectura de códigos QR con una base de datos centralizada, se asegura que la información esté disponible en tiempo real y protegida contra manipulaciones
